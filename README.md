# DSA210-Term-Project
DSA210 Data Science Term Project
Project Overview

This project was developed for the DSA210 Data Science course at Sabancı University. The project focuses on analyzing real-world stock market data through exploratory data analysis, hypothesis testing, and machine learning techniques.

The main objective of the project is to investigate how selected stocks behave relative to the BIST100 market index and determine whether machine learning models can identify meaningful patterns in stock return behavior.

Unlike simplified stock analysis projects, this study uses real financial datasets and incorporates both statistical and predictive analysis methods to evaluate market behavior under different economic conditions.

Datasets Used

The project uses publicly available real-world financial datasets including:

Historical stock prices
BIST100 index data
Daily stock returns
Trading volume information
Market trend indicators

The datasets were collected and processed using Python-based financial data analysis tools.

The BIST100 index was specifically included as the market benchmark instead of using average stock returns, allowing a more realistic comparison with overall market performance.

Exploratory Data Analysis (EDA)

The EDA section includes:

Stock return analysis
Correlation analysis
Volatility analysis
Distribution analysis
Comparative stock performance visualization
Cumulative return analysis
Market trend examination

Several visualizations such as line plots, histograms, boxplots, and correlation heatmaps were generated to better understand stock market behavior and identify relationships between variables.

The project also evaluates how market conditions and volatility influence stock returns over time.

Hypothesis Testing

The project includes statistical hypothesis testing to evaluate whether relationships between stock returns and market performance are statistically significant.

Null Hypothesis (H0)

There is no statistically significant relationship between selected stock returns and the BIST100 market index.

Alternative Hypothesis (H1)

There is a statistically significant relationship between selected stock returns and the BIST100 market index.

Statistical testing methods were applied to support the analytical findings with quantitative evidence.

Machine Learning Analysis

Machine learning models were implemented to analyze and predict stock-related patterns using historical financial data.

Models Used
Linear Regression
K-Nearest Neighbors (KNN)
ML Workflow
Data preprocessing
Feature engineering
Train-test split
Model training
Prediction analysis
Performance evaluation

The machine learning section evaluates the strengths and limitations of predictive modeling in volatile financial markets.

Key Findings
Some stocks demonstrated stronger relationships with market movements than others.
Market volatility significantly affected stock return patterns.
Machine learning models produced varying prediction performance depending on market conditions and selected features.
BIST100 provided a more meaningful benchmark compared to average stock comparisons.
Financial market behavior showed both predictable and highly volatile characteristics during different periods.
